# Predictive Modeling
STA380 is a summer predictive modeling course for the Master's in Business Analytics program at UT Austin. There were 2 assignments for the course:
   - [Exercise 1](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise1)
      * [Exploratory Analysis on Vote Undercount in Georgia](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise1#exploratory-analysis-on-vote-undercount-in-georgia)
      * [Evaluating ETF Portfolios with Bootstrapping](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise1#evaluating-etf-portfolios-with-bootstrapping)
      * [Distinguishing Wines with Clustering and PCA](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise1#distinguishing-wines-with-clustering-and-pca)
      * [Market Segmentation with PCA](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise1#market-segmentation-with-pca)
      
   - [Exercise 2](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise2)
      * [Visualization of Flights at ABIA](https://github.com/juliaawu/sta380-predictive-modeling/tree/master/Exercise2#flights-at-abia)
      * [Author Attribution using Naive Bayes and Principal Component Regression](https://github.com/juliaawu/sta380-predictive-modeling/blob/master/Exercise2/README.md#author-attribution)
      * [Association Rule Mining with Groceries](https://github.com/juliaawu/sta380-predictive-modeling/blob/master/Exercise2/README.md#association-rule-mining-with-groceries)
